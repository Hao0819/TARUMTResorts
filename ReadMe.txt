TARUMT Resorts - Reservation & Room Optimization System
BMCS2063 Data Structures and Algorithms
Team Members: Lim Jun Hao, Gan Koh Jun, Tan Keng Ting, Brian Lee Kit Mun, Gary Khor

=== HOW TO RUN ===
1. Open the project in NetBeans (Apache NetBeans IDE 30 or later)
2. Ensure JDK 11+ is installed
3. Run TARUMTResorts.java (contains the main method)
4. Follow the on-screen console menu to navigate between modules
